# C# Multiple Variables

To declare more than one variable of the same type, use a comma-separated list:

**Example:**

```cs
int x = 5, y = 6, z = 50;
Console.WriteLine(x + y + z);
```

You can also assign the **same value** to multiple variables in one line:

**Example:**

```cs
int x, y, z;
x = y = z = 50;
Console.WriteLine(x + y + z);
```
